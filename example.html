Welcome to TeletextJS

TeletextJS recreates the nostalgic 1980s teletext experience on the web — complete with retro visuals, blocky mosaic graphics, and real-time clock display.

Features of TeletextJS:

  • Authentic 1980s-style teletext look and feel
  • Create colorful mosaic art using the <ttx-mosaic> element
  • Supports rich HTML-like syntax for formatting and color tags
  • Built-in clock and datetime display with customizable layout
  • Keyboard navigation and clickable page numbers
  • Multiple subpages per page with fully custom content

Explore the demo using page links below or navigate pages and subpages with the arrow keys!

Getting Started

To use TeletextJS, include the core script inside your <head> section as follows: 

<script src="js/teletext.js" defer></script>

Then initialize TeletextJS at the bottom of your HTML page using this code: 

<script>
window.addEventListener('DOMContentLoaded', () => {
  Teletext.init(pages, {
    // linkUnderline: false,
    // titleAlign: 'left'
  });
});
</script>

You can insert any global configuration options inside the second argument of Teletext.init(). These options are optional and act as page-wide global settings for this particular HTML page or app. Every parameter defined here automatically applies to all Teletext pages and subpages, unless a specific page overrides them with its own settings.

This is a reliable and fully supported method — in fact, almost every available setting (such as colors, text sizes, header styles, buttons, or behavior flags) can be defined globally in this initialization block. If you omit a setting, TeletextJS automatically falls back first to the values defined in your config.js file, and then to the built-in master defaults in teletext.js.

Configuration: config.js

All user-editable configuration options are located in assets/js/config.js. This file provides a structured way to modify global TeletextJS settings easily without touching the main library files.

It includes adjustable parameters for core behavior, layout, color themes, typography, and visual effects. For example, you can set the default page number, base font size, aspect ratio, or header appearance: 

window.TeletextConfig = {
  defaultPage: 100,
  aspectRatio: 4 / 3,
  baseSize: 22,
  headerBackground: "blue",
  linkUnderline: false,
  enableKeyboard: true
};

These values automatically apply when teletext.js loads, ensuring consistent behavior across every page that uses the library. The config.js file is the recommended place for setting up global site-wide defaults — it keeps the code clean and prevents duplication between multiple HTML pages.

Additional Information & Documentation

Below is a minimal example of defining Teletext pages using JavaScript: 

const pages = {
  100: {
    title: "Home Page",
    content: [
      `[h1]Welcome[/h1]
      [block-blue]This is a TeletextJS page[/block-blue]`
    ]
  }
};

You can include mosaics, color blocks, or even raw HTML-like syntax directly in your page content. TeletextJS automatically parses and renders these elements with an authentic retro teletext look and feel.
